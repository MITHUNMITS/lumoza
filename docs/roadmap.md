# Roadmap

## Current Stage

Lumoza has completed Phase 1 foundation and is now in Phase 2 implementation.

Current full product progress: 31%

## Phases

### Phase 1: Foundation

Goal:

- Desktop shell, projects, SQLite, scans, thumbnails, progress, and operational stability

### Phase 2: Fast AI Engine

Goal:

- Blur, exposure, contrast, resolution, duplicates, bursts, and fast scoring

### Phase 3: Face Intelligence

Goal:

- Face detection, embeddings, clustering, people management, and priority marking

### Phase 4: Smart Selection Engine

Goal:

- Final ranking, refiltering, protections, coverage logic, and low-confidence review

### Phase 5: Professional Polish

Goal:

- Storytelling, aesthetic intelligence, advanced review UX, and manual override refinement

### Phase 6: Production Distribution

Goal:

- Packaging, model management, optimization, portability, and resilience improvements

### Optional Phase 7: Future Ecosystem

Goal:

- Video, mobile companion, optional cloud sync, and wider creative integrations

## Phase Discipline

Every phase should be completed and reviewed before the next phase begins.


## Completion Model

This percentage is full-program progress, not phase-local progress.

Current weighted estimate: `31%` overall.

Interpretation:
- Phase 1: 100% complete
- Phase 2: early implementation in progress
- Phases 3-6: largely unstarted
- Launch-critical cross-cutting work: mostly still open
